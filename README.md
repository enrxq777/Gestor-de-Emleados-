Sistema de Gestión de Tienda de Almacenes
Sistema de administración para tiendas y almacenes desarrollado en Visual Basic .NET, orientado a la gestión de inventario, empleados, ventas y control de almacenes mediante un diseño orientado a objetos.
Descripción
Este proyecto implementa un sistema de gestión para tiendas con múltiples almacenes, permitiendo el control de:
Inventario y productos por almacén
Empleados (administradores, operarios, vendedores)
Cálculo de nómina y pagos
Registro de ventas y movimientos de stock
Validaciones de datos centralizadas (RFC, nombres, salarios, turnos, etc.)
El sistema utiliza herencia, polimorfismo y encapsulamiento para modelar las distintas entidades del negocio.
Tecnologías utilizadas
Lenguaje: Visual Basic .NET
Paradigma: Programación Orientada a Objetos (POO)
